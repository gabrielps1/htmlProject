Ferramenta utilizada para trabalhar com HTML

IDE que vamos utilizar para trabalhar com HTML será a Visual Studio Code. Com o Visual Studio Code instalado, vamos instalar um plugin chamado LIVE SERVER e EMMET.

LIVE SERVER, vai nos permitir executar o projeto em um navegador.

EMMET vai acelerar nosso processo de escrita no desenvolvimento dos codigos.

Trabalhando com inspetor de elementos

O que é o inspetor de elementos?

O inspetor de elementos é uma ferramenta usada através do navegador que nos permite visualizar o codigo HTML/CSS/JAVASCRIPT entre outras informações importantes de uma aplicação web.

Como abrir o inspetor de elementos?

Basta apertar a tecla F12 do teclado no navegador que irá abrir um console.

Estrutura básica do HTML

Quando precisamos criar um nome padrão para dizer que essa vai ser nossa pagina principal do projeto, damos o nome de INDEX.HTML

Quais são as 3 tags principais do HTML?

Uma pagina HTML consiste em 3 TAGS, sendo elas: HTML, HEAD, BODY.

Tudo que tiver entre HTML vai ser uma pagina HTML, um arquivo HTML.

Tudo que tiver entre BODY é o que de fato vai aparecer para o usuario

Tudo que tiver entre o HEAD é o que carrega antes de exibir para o usuario

Falando sobre TAGS

Aqui cito duas tags HTML. Uma delas é a 'I' que deixa o texto em italico e a TAG strong, que deixa o texto em negrito

Atributo das TAGS

Atributos são propriedades que uma tag pode ser. O atributo type é um atributo muito usado. Com esse atributo você pode atribuir valores para ele e esse valor vai criar diferentes campos de dados

Manuseando texto

As principais tags para trabalhar com texto são: H1,H2,H3. Cada um desses é um nivel de formatação de texto. Usando eles o texto pode ficar maior ou menor

A tag P representa um paragrafo

A tag blackquote é usado para fazer citações

A tag sup é usado para deixar numeros ou letras em cima da palavra desejada.

A tag mark deixa o texto em marca texto

A tag u deixa o texto sublinhado
